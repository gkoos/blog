[Ffetch](https://www.npmjs.com/package/@fetchkit/ffetch) v5.1.0 is out!

Ffetch is a lightweight, production-ready TypeScript-first drop-in replacement for native fetch.

This release introduces two new convenience plugins:

- `requestShortcutsPlugin` for HTTP method shortcuts on the client (`get`, `post`, `put`, `patch`, `delete`, `head`, `options`)
- `responseShortcutsPlugin` for response parsing shortcuts on the request promise (`json`, `text`, `blob`, `arrayBuffer`, `formData`)

It also includes a full documentation refresh across README, API reference, plugin architecture, examples, migration guide, and docs index.

GitHub: [fetch-kit/ffetch](https://github.com/fetch-kit/ffetch)

Any feedback or contributions are welcome in the GitHub repository.
